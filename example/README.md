# screen_launcher example

A runnable Flutter app showing how to use `screen_launcher`.

The example demonstrates:

- defining `LaunchableScreen` entries
- opening the launcher from a button with `ScreenLauncher.open(...)`
- showing a draggable overlay bubble with `ScreenLauncher.showBubble(...)`
- launching routes through the app's own `Navigator.pushNamed(...)` callback

Run it from this directory:

```bash
flutter run
```
